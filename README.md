# Gådjå Etiopisk Restaurant

Website for Gådjå Etiopisk Restaurant, an Ethiopian restaurant located in Stavanger, Norway.

## About

A single-page website featuring:
- Restaurant information and opening hours
- Menu with PDF download
- Customer reviews
- Contact information with Google Maps
- Online ordering links (Wolt, Foodora)
- Bilingual support (Norwegian / English)

## Technology

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework (via CDN)
- **Vanilla JavaScript** - No frameworks, pure JS for interactivity
- **Google Fonts** - Playfair Display (headings) and Open Sans (body)

## Structure

```
├── index.html          # Main website
├── assets/
│   ├── css/
│   │   └── styles.css  # Custom styles
│   ├── img/            # Images and menu PNGs
│   └── pdf/            # Menu PDFs
└── js/
    └── main.js         # JavaScript functionality
```

## Local Development

Simply open `index.html` in a browser. No build step required.
